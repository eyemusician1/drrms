Pre-requisites
- Download MongoDB Compass
- Set a connection, usually like this (mongodb://localhost:27017)
- Create .env for yourself in the backend

MONGO_URI=mongodb://localhost:27017
DB_NAME=drrms_db
JWT_SECRET=this-is-a-super-secret-key-that-is-at-least-32-chars
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=60
REFRESH_TOKEN_EXPIRE_DAYS=7
APP_ENV=development
APP_NAME=DRRMS API
APP_VERSION=1.0.0
ALLOWED_ORIGINS='["http://localhost:5173"]'
RATE_LIMIT_PER_MINUTE=100
AUTH_RATE_LIMIT_PER_MINUTE=10
SECURE_HEADERS=false

Backend
- cd backend
- pip install -r requirements.txt
- uvicorn main:app --reload
if doesn't works:
- python -m uvicorn main:app --reload

To check it works, go to MongoDB Compass if the drrms_db automatically created

To create your own data,
- tyoe in browser "http://localhost:8000/api/docs"
- Find POST /api/v1/auth/register and Execute
- Go back to MongoDB Compass and refresh, you'll see the officers_table (collection)
